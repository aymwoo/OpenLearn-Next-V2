# Implementation Plan - Task 12-01-03

## Goal
Implement the `SocketBridge` reference-counting network subscription bridge in `src/mfe/MfeContextProvider.tsx` and the `MfeEventBusWrapper` in `src/mfe/MfeLoaderCore.tsx`.

## Proposed Changes

### 1. `src/mfe/MfeContextProvider.tsx`
- Implement class `SocketBridge` with constructor parameters `socketService` and `hostEventBus`.
- Implement `register` and `unregister` with reference counting (incrementing/decrementing subscriptions, and registering/unregistering single socket listener).

### 2. `src/mfe/MfeLoaderCore.tsx`
- Import `PlatformEvent` and `EventBus` from `../../packages/core/event-bus`.
- Implement class `MfeEventBusWrapper`.
- Implement `subscribe` with support for `server:` events (passing to `socketBridge`), returning unsubscribe.
- Implement `publish` wrapping metadata (injecting `source`, `id`, `timestamp`) and delegating `server:` events directly to socket.
- Implement `cleanup` to unsubscribe all active listeners.
