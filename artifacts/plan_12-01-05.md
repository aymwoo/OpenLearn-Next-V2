# Implementation Plan - Task 12-01-05

## Goal
Implement `src/mfe/useMfeEvent.ts` hook and integrate event wrappers/proxies inside `src/mfe/MfeLoaderCore.tsx` loading lifecycle.

## Proposed Changes

### 1. `src/mfe/useMfeEvent.ts`
- Implement custom hook `useMfeEvent(eventType, handler)`.
- Retrieve context via `useMfeContext()`.
- Use a `useEffect` to subscribe to the event type on `infra.eventBus` and return the unsubscribe callback.

### 2. `src/mfe/MfeLoaderCore.tsx`
- Retrieve `infra = useMfeInfraContext()`.
- Define `eventBusWrapperRef = useRef<MfeEventBusWrapper | null>(null)`.
- In the `run()` method of loading effect, when `mod.createMfeApp` is found:
  - Get `socketService` using token `'@openlearn/frontend:ISocketService'`.
  - Instantiate `SocketBridge`, `MfeEventBusWrapper`, and `MfeServiceRegistryProxy`.
  - Create the wrapped `mfeContext` and pass to `mod.createMfeApp(mfeContext)`.
  - Assign `eventBusWrapperRef.current = eventBusWrapper`.
- In `cleanup()` callback, call `eventBusWrapperRef.current?.cleanup()` and set to null.
