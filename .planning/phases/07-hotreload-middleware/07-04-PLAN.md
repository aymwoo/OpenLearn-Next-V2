### Task 1: Hot Reload E2E 集成测试

**Files:**
- `packages/core/plugin-host/__tests__/hot-reload.test.ts` — 新建

**Description:**

创建独立的热重载测试文件，测试完整的 reload 生命周期:

1. **E2E: install → activate → reload → verify new behavior**
   ```
   // 安装 v1（handler 返回 'v1'）
   // 激活 v1
   // 验证: ctx.services.commandBus.execute() 返回 'v1'
   // reload v2 (handler 返回 'v2')
   // 验证: ctx.services.commandBus.execute() 返回 'v2'
   ```

2. **E2E: reload 保留 pluginId 不变**
   ```
   // 安装 → 激活 → reload
   // 验证: pluginId 不变, pluginStates.get(pluginId) === ACTIVE
   ```

3. **E2E: reload 后旧事件订阅被移除**
   ```
   // v1: activate 中订阅 eventBus.subscribe('test.event', handler1)
   // reload v2: activate 中订阅 eventBus.subscribe('test.event', handler2)
   // 发布 'test.event' → 仅 handler2 被调用，handler1 不被调用
   ```

4. **E2E: reload 失败 → 旧版本继续运行**
   ```
   // 安装 v1（handler 返回 'v1'）→ 激活
   // reload v2（语法错误源码）
   // 验证: 抛出 HotReloadActivationError
   // 验证: 旧 handler 仍返回 'v1'
   // 验证: pluginStates.get(pluginId) === ACTIVE（未变）
   ```

5. **E2E: manifest.id 不匹配 → 拒绝 reload**
   ```
   // 安装 plugin-A → 激活
   // reload manifest.id = 'plugin-B' 的新源码
   // 验证: 抛出错误包含 'manifest id mismatch'
   ```

测试使用 vitest + mock EsmLoader:
- Mock `esmLoader.load()` 返回 `{ activate: vi.fn(), deactivate: vi.fn() }`
- 创建测试插件源码字符串（`createTestPluginSource(manifestId, handlerResult)`）
- 每个测试前后清理 PluginHost 内部状态

**Verification:**
- 5 个 E2E 测试全部通过
- 不依赖真实文件系统
- 不依赖 chokidar

### Task 3: 压力测试 — 10 次连续 Reload

**Files:**
- `packages/core/plugin-host/__tests__/hot-reload.test.ts` — 追加测试

**Description:**

9. **10 次连续 reload 无内存泄漏**
   ```
   // 安装 v1 → 激活
   // 循环 10 次: reload(v{i} → v{i+1})
   // 验证:
   //   - pluginStates.size === 初始值（不变）
   //   - pluginInstances.size === 初始值（不变）
   //   - 旧版本 deactivate 被调用 10 次
   //   - ResourceTracker 中该 pluginId 的资源数组长度为 0（每次 reload 前已清理）
   ```

10. **10 次连续 reload 无 CPU 增长**
    ```
    // 测量每次 reload 的耗时
    // 验证: 第 10 次 reload 耗时 ≤ 第 1 次 reload 耗时 * 1.5（允许 50% 波动）
    // 移除明显的递增趋势
    ```

测试使用 vitest 的性能计时:
```typescript
const durations: number[] = [];
for (let i = 0; i < 10; i++) {
  const start = performance.now();
  await pluginHost.reloadPlugin(pluginId, createTestPluginSource(`v${i + 2}`, `result${i + 2}`));
  durations.push(performance.now() - start);
}
// 检查: durations 无明显递增趋势
// durations[9] <= durations[0] * 1.5
```

**Verification:**
- 10 次 reload 后无状态泄漏
- 性能无明显退化
- ResourceTracker 清理正确
