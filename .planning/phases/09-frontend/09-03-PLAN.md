    - This file's content is used as the Worker bootstrap code. It runs inside the Web Worker context.
    - Exports nothing directly — it's a self-contained script loaded via `new Worker(blobUrl, { type: 'module' })`.

    - Content (mirrors backend generateBootstrapCode at worker-manager.ts lines 247-531 but for browser):

      ```typescript
      // Self-contained Worker bootstrap for browser Web Worker
      // No imports allowed — runs in isolated Worker context

      const pendingCalls = new Map();

      self.onmessage = async (event: MessageEvent) => {
        const msg = event.data;

        // 1. Forwarded event dispatch
        if (msg?.type === 'event') {
          const handlers = eventSubscriptions.get(msg.subId);
          if (handlers) {
            for (const handler of handlers) {
              try { handler(msg.event); } catch (e) {
                console.error('[Worker] Event handler error:', e);
              }
            }
          }
          return;
        }

        // 2. RPC result/error dispatch (has invokeId and pending call)
        if (msg?.invokeId && pendingCalls.has(msg.invokeId)) {
          const pending = pendingCalls.get(msg.invokeId);
          pendingCalls.delete(msg.invokeId);
          if (msg.type === 'error') {
            const err = new Error(msg.message);
            err.name = msg.code || 'RpcError';
            err.stack = msg.stack;
            pending.reject(err);
          } else if (msg.type === 'result') {
            pending.resolve(msg.value);
          }
          return;
        }

        // 3. Activate
        if (msg.type === 'activate') {
          try {
            // Create service proxies for all declared tokens
            const services = {};
            for (const token of msg.serviceTokens) {
              services[token] = createServiceProxy(token, msg.serviceTokens);
            }

            // Load plugin via Blob URL import()
            const blob = new Blob([msg.pluginCode], { type: 'text/javascript' });
            const url = URL.createObjectURL(blob);
            let mod;
            try {
              mod = await import(url);
            } finally {
              URL.revokeObjectURL(url);
            }
            const plugin = mod.default || mod;

            if (typeof plugin.activate !== 'function') {
              self.postMessage({ type: 'error', message: 'Plugin has no activate function' });
              return;
            }

            // Build PluginContext with proxy services + UI proxy
            const ctx = {
              services,
              pluginId: msg.manifest?.id || 'unknown',
              manifest: msg.manifest,
              ui: {
                registerExtensionPoint: (...args) => {
                  self.postMessage({ type: 'invoke', invokeId: crypto.randomUUID(),
                    token: '@openlearn/frontend:IUIService', method: 'registerExtensionPoint', args });
                },
                unregisterExtensionPoint: (...args) => {
                  self.postMessage({ type: 'invoke', invokeId: crypto.randomUUID(),
                    token: '@openlearn/frontend:IUIService', method: 'unregisterExtensionPoint', args });
                },
              },
            };

            await plugin.activate(ctx);
            self.postMessage({ type: 'activated' });

            // 4. Handle deactivate request (register after activation)
            self.onmessage = async (dmsg) => {
              if (dmsg.type === 'deactivate-request') {
                try {
                  if (typeof plugin.deactivate === 'function') {
                    await plugin.deactivate();
                  }
                } finally {
                  pendingCalls.clear();
                  eventSubscriptions.clear();
                  self.postMessage({ type: 'deactivated' });
                }
              }
            };
          } catch (err) {
            self.postMessage({
              type: 'error',
              message: err?.message || String(err),
              stack: err?.stack || '',
            });
          }
        }
      };

      // Service proxy factory
      function createServiceProxy(token, args) {
        return new Proxy({}, {
          get(_target, method) {
            return (...args) => {
              const invokeId = crypto.randomUUID();
              return new Promise((resolve, reject) => {
                pendingCalls.set(invokeId, { resolve, reject });
                self.postMessage({ type: 'invoke', invokeId, token, method: String(method), args });
              });
            };
          },
        });
      }

      const eventSubscriptions = new Map();
      ```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Main thread ↔ Web Worker | Untrusted plugin code in Worker communicates via postMessage structured clone protocol |
| Worker → ServiceHost RPC | Worker sends invoke messages; ServiceHost resolves frontend services and executes methods |
| Socket.IO event → Worker | Events from server forwarded to Worker via postMessage |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-07 | Tampering | Worker postMessage data injection | mitigate | ServiceHost validates message.type before dispatch; unknown types silently ignored per backend pattern |
| T-09-08 | Elevation of Privilege | RPC proxy method injection | mitigate | Capability check: empty manifestCapabilities blocks non-'get' methods per Phase 5 pragmatic rule |
| T-09-09 | Denial of Service | Worker resource exhaustion | mitigate | BrowserWorkerManager.MAX_WORKERS = 32, enforce at createWorker time |
| T-09-10 | Tampering | Worker event subscription leak on termination | mitigate | terminateWorker calls serviceHost.dispose() before Worker termination per backend pattern (finally block) |
| T-09-11 | Information Disclosure | Worker accessing DOM/localStorage directly | mitigate | D-10 enforcement: Worker runs in isolated context, no DOM/localStorage/fetch access. All operations go through ServiceProxy RPC |
| T-09-12 | Denial of Service | Blob URL leak in createWorker | mitigate | Use try/finally with URL.revokeObjectURL() after activation (success or failure) |
</threat_model>
