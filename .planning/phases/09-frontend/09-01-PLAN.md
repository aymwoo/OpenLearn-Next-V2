## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| FrontendPluginHost ↔ ServiceRegistry | Registry holds references to frontend services; untrusted code should not mutate the registry |
| Plugin activate() ↔ BrowserEsmLoader | Plugin source code loaded via Blob URL + import() runs in main-thread context |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-09-01 | Tampering | FrontendServiceRegistry.register() | mitigate | Throw on duplicate registration per D-08 mirror; prevent service replacement without explicit unregister |
| T-09-02 | Denial of Service | Blob URL leak in activatePlugin | mitigate | Always use try/finally with URL.revokeObjectURL() after import() resolves, matching BrowserEsmLoader pattern |
| T-09-03 | Information Disclosure | IStorageService cross-plugin key collision | mitigate | Prefix localStorage keys with `edu_os_plugin:{pluginId}:` to namespace per plugin |
</threat_model>
