## 2. 跨阶段集成与端到端链路 (Integration & E2E Flows)

- **动态加载与降级链路 (Flow-01)**：当子应用服务断开或端口不通时，`MfeLoader` 捕获异常，阻断异常级联，展示降级提示，而宿主面板其他部分保持完好，验证了 Fail-Safe 容灾设计。
- **DI 注入与通信链路 (Flow-02)**：子应用通过 `MfeServiceRegistryProxy` 成功获取到宿主的 `ISocketService` 实例，实现与 EventBus 的完全协同。
- **样式隔离与沙箱验证 (Flow-03)**：Vite MF 2.0 打包产物 CSS 不含任何针对 `*`, `body`, `html` 的重置定义，类名全部自带 `wb:`/`cw:` 前缀，无布局泄漏冲突。
